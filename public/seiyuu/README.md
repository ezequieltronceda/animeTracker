# Imágenes de seiyuus

Colocá acá los dos recortes usados por el contador de seiyuus (ver `lib/anime-constants.ts`).
Nombres exactos (el código apunta a estas rutas):

- `koyasu.png` → DIO (JoJo's Bizarre Adventure) para **Takehito Koyasu**
- `hanazawa.png` → Mayuri Shiina (Steins;Gate) para **Kana Hanazawa**

Idealmente cuadradas y centradas en el rostro: se muestran chicas y redondeadas
(~22–26 px, `border-radius: 50%`) en la barra superior, las tarjetas y el detalle.
