# Project Overview

Web app para trackear animes de temporada entre dos usuarios (Eze y Pancho).

## Core features
- Tabla principal con animes por temporada
- Subfilas por usuario con progreso de capítulos
- Estados configurables
- Opinión al terminar
- Integración con Jikan (MAL data)
- Cache en Firestore + revalidación manual
- Drawer para agregar anime
- Modal para detalle

## Stack
- Next.js (App Router)
- Firebase Firestore
- TailwindCSS
- Zustand (estado cliente)
